# Poster Assets

このディレクトリには、`/poster` コマンドで使用される画像アセットを配置してください。

## 必要なファイル

以下の画像ファイルが必要です：

2. **mask.png** - キャラクター画像を合成するためのマスク画像
3. **peaceful.png** - Peaceful国の旗画像
4. **brave.png** - Brave国の旗画像
5. **glory.png** - Glory国の旗画像
6. **freedom.png** - Freedom国の旗画像

## セットアップ

これらの画像ファイルをこのディレクトリ (`data/assets/`) に配置してください。

画像ファイルが用意できていない場合、`/poster` コマンドは適切なエラーメッセージを表示します。

## 注意事項

- 画像は PNG 形式を推奨します
- ファイル名は必ず上記の通りにしてください（大文字小文字を区別します）
- 画像サイズが大きすぎると処理が遅くなる可能性があります
